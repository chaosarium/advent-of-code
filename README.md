## Useful Commands

- Rust
	- `cargo new day01`
	- `cargo run`
- OCaml
	- `dune init day02`
	- `dune exec day02`
- Go
	- create: `mkdir day05`, `cd day05`, `go mod init main`
	- `go run main`
- Python
	- just create a `main.py`
	- `python main.py`
- Haskell
	- make a `main.hs`
	- `ghc main.hs && ./main`
- Typst
	- just create a `main.typ`
	- `typst compile main.typ`
- C
  - just create a `main.c`
  - `gcc main.c -o main && ./main`
