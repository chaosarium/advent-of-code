## Useful Commands

- Rust
	- `cargo new day01`
	- `cargo run`
- OCaml
	- `dune init day02`
	- `dune exec day02`
- Go
	- create: `mkdir day05`, `cd day05`, `go mod init main`
	- `go run main`
- Python
	- just create a `.py`
	- `python main.py`
- Typst
	- just create a `.typ`
	- `typst compile main.typ`
